# A. Точка окупаемости

## Ограничения
- Ограничение времени: 1 секунда
- Ограничение памяти: 64 Мб
- Ввод: стандартный ввод или input.txt
- Вывод: стандартный вывод или output.txt

## Описание
Оперативная память сейчас стоит дорого, и Вася решил организовать ее производство. Всего существует $N$ различных станков для изготовления чипов памяти. Каждый станок описывается тремя числами: $a_i$ — стоимость станка, $b_i$ — затраты на производство одного чипа памяти, $c_i$ — сколько покупатели готовы заплатить за один чип, изготовленный на этом станке.

Точкой окупаемости назовем минимальное целое количество чипов, которое необходимо продать, чтобы покрыть расходы на закупку станка и изготовление этих чипов.

Определите, какой станок нужно купить, чтобы достичь точки окупаемости изготовив как можно меньше чипов.

## Формат ввода
В первой строке вводится число $N$ — количество станков ($1 \le N \le 10^5$).

В каждой из следующих $N$ строк вводится по три целых числа $a_i$, $b_i$, $c_i$ ($1 \le a_i \le 10^9$, $1 \le b_i < c_i \le 10^9$).

## Формат вывода
Если введено отрицательное значение N — выведите -2.

Выведите номер (индекс) оптимального станка (при нумерации станков с единицы). Если оптимальных станков несколько, выведите наименьший номер оптимального станка.

## Примеры
### Пример 1
Ввод
```
2
1 2 3
2 1 3
```
Вывод
```
1
```

### Пример 2
Ввод
```
3
1 2 4
3 1 4
2 2 4
```
Вывод
```
1
```
