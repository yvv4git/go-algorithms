# C. Социальная поддержка

## Ограничения
- Ограничение времени: 1 секунда
- Ограничение памяти: 64 Мб
- Ввод: стандартный ввод или input.txt
- Вывод: стандартный вывод или output.txt

## Описание
Благодаря искусственному интеллекту количество работы у людей сократилось, а количество денег на социальную поддержку возросло. Вася остался совсем без работы и сейчас у него ровно ноль рублей.

Вася узнал о программах социальной поддержки на ближайшие $N$ месяцев. В i-м месяце действует $k_i$ программ социальной поддержки. Для каждой программы $j$ в i-м месяце задано число $a_{ij}$. Если накопления человека не превышают $a_{ij}$, то они увеличиваются до $a_{ij}$ рублей. Если накопления человека превышают $a_{ij}$, программу использовать нельзя. В каждом из $N$ месяцев Вася обязан воспользоваться ровно одной программой, действующей в этом месяце.

Вася не планирует тратить деньги ни в один из $N$ месяцев. Определите, какое максимальное количество рублей может иметь Вася к концу $N$-го месяца.

## Формат ввода
В первой строке вводится число $N$ ($2 \le N \le 1000$) — количество месяцев.

В каждой из следующих $N$ строк вводится описание программ социальной поддержки, действующих в соответствующем месяце. Первое число в описании — $k_i$ ($1 \le k_i \le 1000$) — количество программ социальной поддержки, действующих в этом месяце. Следующие $k_i$ чисел $a_{ij}$ ($1 \le a_{ij} \le 10000$) задают параметры программ.

## Формат вывода
Если введено некорректное значение N выведите -2.

Выведите максимальное количество рублей, которое сможет накопить Вася. Если хотя бы в одном месяце нельзя воспользоваться ни одной программой социальной поддержки, выведите -1.

## Система оценивания
Решения, верно работающие при N = 2, будут получать не менее 40 баллов.

## Примеры
### Пример 1
Ввод
```
2
2 1 40
3 5 6 15
```
Вывод
```
15
```

### Пример 2
Ввод
```
2
2 4 5
2 3 2
```
Вывод
```
-1
```

## Примечания
В первом примере в первом месяце Вася выбирает программу увеличение капитала до 1. 
Если он выберет программу увеличения капитала до 40, то не сможет воспользоваться ни одной программой во втором месяце.
