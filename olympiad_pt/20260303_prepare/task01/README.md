# A. Подготовка к олимпиаде

## Ограничения
- Ограничение времени: 1 секунда
- Ограничение памяти: 256 Mb
- Ввод: стандартный ввод или input.txt
- Вывод: стандартный вывод или output.txt

## Описание
Федя и Кирилл готовятся к олимпиаде и решают задачи.

Федя решил $e_f$ простых задач, $m_f$ средних и $h_f$ сложных. Кирилл решил $e_k$ простых задач, $m_k$ средних и $h_k$ сложных.

На решение простой задачи каждый из них тратит три минуты, средней — 20 минут, а сложной — два часа. Определите, кто из них потратил больше времени на подготовку к олимпиаде.

## Формат ввода
В первой строке вводятся три целых числа $e_f, m_f, h_f$ — количество простых, средних и сложных задач, которые решил Федя ($0 \le e_f, m_f, h_f \le 1000$).

Во второй строке вводятся три целых числа $e_k, m_k, h_k$ — количество простых, средних и сложных задач, которые решил Кирилл ($0 \le e_k, m_k, h_k \le 1000$).

## Формат вывода
Если Федя потратил больше времени на решение задач, то выведите `Fedya`.

Если Кирилл потратил больше времени на решение задач, то выведите `Kirill`.

Если же они потратили одинаковое количество времени, то выведите `Draw`.

## Примеры
### Пример 1
Ввод
```
6 0 0
0 1 0
```
Вывод
```
Kirill
```

### Пример 2
Ввод
```
2 2 2
1 1 1
```
Вывод
```
Fedya
```

### Пример 3
Ввод
```
0 12 0
0 0 2
```
Вывод
```
Draw
```
