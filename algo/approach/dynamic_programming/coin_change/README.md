# Coin Change

## About

Задача: даны монеты разных номиналов (coins) и сумма (amount).
Нужно найти минимальное количество монет, чтобы набрать эту сумму.
Каждую монету можно использовать неограниченное количество раз.
Если набрать сумму невозможно — вернуть -1.

## Solution

dp[i] — минимальное количество монет для суммы i.

Для каждой суммы i перебираем все монеты:
dp[i] = min(dp[i], dp[i - coin] + 1)

dp[0] = 0 (для нулевой суммы монет не нужно)
Остальные dp[i] инициализируем как +inf (math.MaxInt).

Если dp[amount] остался +inf — сумма недостижима, возвращаем -1.

## Example

coins = [1, 2, 5], amount = 11
dp[11] = 3 (5 + 5 + 1)

## Complexity

Time: O(N * S), где N — количество монет, S — сумма.
Space: O(S) — массив dp размера amount+1.
