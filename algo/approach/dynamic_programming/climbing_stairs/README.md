# Climbing Stairs

## About

Задача: есть лестница из n ступенек. За шаг можно подняться на 1 или 2 ступеньки. Сколько всего способов добраться до вершины?

## Solution

dp[i] — количество способов подняться на i-ю ступеньку.

На i-ю ступеньку можно прийти либо с (i-1), либо с (i-2). Значит:
dp[i] = dp[i-1] + dp[i-2]

Начальные значения:
dp[0] = 1 (стоять внизу — 1 способ)
dp[1] = 1 (только шаг на 1)

Фактически это та же последовательность Фибоначчи, но в задаче про количество способов.

## Example

n=4: [1+1+1+1] [1+1+2] [1+2+1] [2+1+1] [2+2] = 5 способов.

## Complexity

Time: O(n)
Space: O(n) — можно сократить до O(1), храня только два последних значения.
