# F. Ход конём - 2

## Ограничения
- Ограничение времени: 1 секунда
- Ограничение памяти: 64 Mb
- Ввод: стандартный ввод или input.txt
- Вывод: стандартный вывод или output.txt

## Описание
Дана прямоугольная доска размером **N×M** (N строк и M столбцов).
В клетке **(1, 1)**, соответствующей левому верхнему углу доски, находится конь.
Требуется определить количество различных маршрутов, по которым конь может попасть в клетку **(N, M)** — правый нижний угол доски.

За один ход конь может переместиться в одну из следующих клеток:
- на **2 столбца вправо и 1 строку вниз**;
- на **1 столбец вправо и 2 строки вниз**;
- на **2 столбца вправо и 1 строку вверх**.

Ходы, выходящие за пределы доски, запрещены.

При этом конь может ходить следующим образом:

![Knight moves](image.png)

Необходимо определить, сколько существует различных маршрутов, ведущих из левого верхнего в правый нижний угол.

## Формат ввода
Входной файл содержит два натуральных числа N и M (1 ≤ N, M ≤ 50).

## Формат вывода
В выходной файл выведите единственное число — количество способов добраться конём до правого нижнего угла доски.

## Примеры
### Пример 1
Ввод
```
4 4
```
Вывод
```
2
```

### Пример 2
Ввод
```
2 3
```
Вывод
```
1
```
