# Knapsack

## About

Задача 0/1 Knapsack (рюкзак). Дано N предметов, у каждого вес w_i и ценность v_i. Нужно набрать предметы суммарным весом не больше W, чтобы суммарная ценность была максимальной. Каждый предмет можно взять не больше одного раза.

## Solution

`dp[i][w]` — макс. ценность для первых i предметов при вместимости w.

Формула:
`dp[i][w] = max(dp[i-1][w], dp[i-1][w - weight[i-1]] + value[i-1])`, если предмет помещается.
Иначе `dp[i][w] = dp[i-1][w]`.

Ответ: `dp[N][W]`.

## Complexity

Time: `O(N * W)`
Space: `O(N * W)` — можно сократить до `O(W)`, храня только предыдущую строку.
