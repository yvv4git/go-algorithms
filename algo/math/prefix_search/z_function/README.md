# Z-Function (Z-функция)

## About

Z-функция — это вспомогательная функция, используемая в алгоритмах поиска подстрок
и обработки строк. Является прямым аналогом префикс-функции: обе выражаются
друг через друга за O(n).

Для строки s длины n Z-функция z[i] определяется как длина наибольшего совпадения
префикса s и подстроки, начинающейся в позиции i (то есть s[0..] и s[i..]).
z[0] обычно принимается равным 0 (или длине строки — в разных определениях).

## Основные характеристики

- **Временная сложность**: O(n)
- **Пространственная сложность**: O(n)
- **Тип данных**: Строка (последовательность символов)

## Принцип работы

1. Поддерживается отрезок \[l, r), который является суффиксом строки и одновременно
   её префиксом (самый правый z-box).
1. Для i от 1 до n − 1:
   - Если i < r, начальное значение z[i] = min(r − i, z[i − l]).
   - Наивное расширение, пока символы совпадают.
   - Если i + z[i] > r, обновляем l = i, r = i + z[i].

Благодаря z-box каждый символ строки сравнивается не более одного раза,
что даёт O(n).

### Пример

```text
Строка:  A B A C A B A B
z:       0 0 1 0 3 0 2 0

Разбор:
i=2: z[2]=1 — "A" совпадает с префиксом "A"
i=4: z[4]=3 — "ABA" совпадает с префиксом "ABA"
i=6: z[6]=2 — "AB" совпадает с префиксом "AB"
```

## Где используется

- **KMP**: Z-функция как альтернатива префикс-функции (дают один результат)
- **Поиск периода строки**: через Z-функцию проще, чем через префикс-функцию
- **Поиск подстроки**: конкатенация pattern + '$' + text и вычисление Z-функции
- **Анализ строк**: поиск повторов, палиндромов, уникальных подстрок

## Применение: поиск подстроки через Z-функцию

Склеиваем pattern + '$' + text и вычисляем Z-функцию. Если z[i] == m — найдено
вхождение на позиции i − m − 1:

```text
Паттерн: "AB"
Текст:   "ABABD"
Склейка: "AB$ABABD"
z:        0 0 0 2 0 2 0 0
              ↑   ↑ z=2 → вхождение на позиции 0 и 2
```

## Сравнение с аналогами

| Функция | Назначение | Сложность | Память |
|---------|-----------|-----------|--------|
| Префикс-функция | Длина общего префикса/суффикса для каждого префикса | O(n) | O(n) |
| Z-функция | Длина совпадения префикса и подстроки для каждой позиции | O(n) | O(n) |
| Hash (rolling) | Хеш подстроки | O(n) | O(1) |

Z-функция и префикс-функция взаимозаменяемы. Z-функция проще для поиска подстрок
(не нужно строить автомат), префикс-функция компактнее для пошаговой обработки.
