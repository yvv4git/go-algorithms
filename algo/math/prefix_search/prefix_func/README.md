# Prefix Function (π-функция)

## About

Префикс-функция (π-функция) — это вспомогательная функция, используемая
в алгоритме Кнута-Морриса-Пратта (KMP) и других алгоритмах поиска подстрок.

Для строки s длины n префикс-функция π[i] определяется как длина наибольшего
собственного (не равного всей подстроке) префикса подстроки s[0..i],
который одновременно является её суффиксом.

Если префикс-функция равна 0, это означает, что у данного префикса нет
собственного префикса, совпадающего с суффиксом.

## Основные характеристики

- **Временная сложность**: O(n)
- **Пространственная сложность**: O(n)
- **Тип данных**: Строка (последовательность символов)

## Принцип работы

1. π[0] = 0 всегда (у строки из одного символа нет собственного префикса).
1. Для i от 1 до n-1:
   - Берём j = π[i-1] — длину префикса, который уже совпал с суффиксом.
   - Пока j > 0 и s[i] ≠ s[j], переходим к π[j-1] (пробуем меньший префикс).
   - Если s[i] = s[j], увеличиваем j на 1.
   - π[i] = j.

### Пример

```text
Строка:  A B A C A B A B
π:       0 0 1 0 1 2 3 2
            ↑       ↑
   π[2]=1       π[5]=2 — "AB" является префиксом и суффиксом для "ABACAB"
```

## Где используется

- **KMP**: построение LPS-массива для поиска одного паттерна
- **Поиск периода строки**: если n mod (n - π[n-1]) = 0, то строка периодична
- **Ахо-Корасик**: fail-ссылки для бора — аналог префикс-функции на дереве
- **Автоматы**: построение конечного автомата по префикс-функции
- **Сжатие**: выделение повторяющихся блоков в строке

## Применение: поиск периода строки

Если n - π[n-1] делит n нацело, то длина периода строки равна n - π\[n-1\]:

```text
Строка:  "ABABAB"  (n=6)
π:        [0 0 1 2 3 4]
Период:   n - π[5] = 6 - 4 = 2  →  "AB"
Проверка: "AB" × 3 = "ABABAB"  ✓
```

## Сравнение с аналогами

| Функция | Назначение | Сложность | Память |
|---------|-----------|-----------|--------|
| Префикс-функция | Длина общего префикса/суффикса для каждого префикса | O(n) | O(n) |
| Z-функция | Длина совпадения префикса и подстроки для каждой позиции | O(n) | O(n) |
| Hash (rolling) | Хеш подстроки | O(n) | O(1) |

Префикс-функция и Z-функция взаимозаменяемы (выражаются друг через друга за O(n)).
