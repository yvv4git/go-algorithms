# D. Гласно-согласный поиск

Даны текст $T$ и слово $S$.

В слове $S$ разрешается менять любые гласные буквы на любые другие гласные (гласными считаются буквы $a$, $e$, $i$, $o$, $u$ и $y$), а также любые согласные — на любые другие согласные.

Найдите все позиции, с которых начинаются вхождения $S$ в $T$ с учётом возможных замен.

## Формат ввода

Первая строка содержит последовательность $T$ ($1 \le |T| \le 2 \cdot 10^6$), состоящую из строчных латинских букв, — текст.

Вторая строка содержит последовательность $S$ ($1 \le |S| \le 2 \cdot 10^6$), состоящую из строчных латинских букв, — слово, которое ищется в тексте.

## Формат вывода

Выведите одно или более целых чисел — позиции начала вхождений $S$ в $T$ с учётом возможных замен букв.

Если $S$ не содержится в $T$, выведите число $-1$.

## Пример 1

Ввод

```text
abracadabra
abr
```

Вывод

```text
0 7
```

## Пример 2

Ввод

```text
notsoverylongtext
word
```

Вывод

```text
0 9 13
```
