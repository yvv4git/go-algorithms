# B. Цензура

Посчитайте, сколько строк длины m над алфавитом из n символов не содержат ни одной подстроки из заданного множества «запрещенных» строк.

## Формат ввода

В первой строке через пробел даны три целых числа:

- n (1 ≤ n ≤ 100) — количество символов в алфавите
- m (1 ≤ m ≤ 100) — длина искомых строк
- p (0 ≤ p ≤ 10) — количество «запрещенных» подстрок

Следующая строка содержит ровно n символов с кодами больше 32 — буквы алфавита. Далее идут p «запрещенных» строк, по одной на строке. Длина каждой такой строки не превосходит min(m, 10) символов. Строки целиком состоят из символов алфавита.

## Формат вывода

В единственной строке выведите ответ на задачу по модулю 1 000 000 007.

## Пример

Ввод

```text
2 3 1
ab
bb
```

Вывод

```text
```
